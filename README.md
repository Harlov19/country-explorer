# Country Explorer

**Country Explorer** is a web application that allows users to search for any country and view comprehensive details such as its capital city, population, area, languages, local times, and location on an interactive map.

## Features
- Live search with real-time suggestions  
- Interactive map rendered using **Leaflet** and **OpenStreetMap**  
- Display of real-time local times for each timezone  
- Theme toggle between light and dark modes  
- Optimized and cached API requests for faster performance  

## Technologies Used
- **HTML5**, **CSS3**, **JavaScript (ES6+)**  
- **REST Countries API** for fetching country data  
- **Leaflet.js** for map integration
